Module Cardeffects

module Cardeffects: sig .. end
val shuffle : Gamestate.game -> unit

shuffle g is the same as Gamestae.deck, but all elements are reordered randomly

val prophet_llama : Gamestate.game -> unit

Prophet_llama g Rearranges the deck from g using 2 inputs from the player who played it (the current player in g)

val rank_card : Deck.card -> int

rank_card card takes a card and returns the value of the card, which corrresponds to how good the card is, higher is better

val compare_card_use : Deck.card -> Deck.card -> int

compare_card_use card1 card2 compares card1 with card2. If card1 has a better rank than card2, compare_card_use returns 1, if card2 has a better rank than card1, compare_card_use returns -1, otherwise 0

val skip : Gamestate.game -> unit

skip player does a pass action, without the drawing requirement

val attack : Gamestate.game -> unit

attack player forces the player to take two turns

val favor : Gamestate.game -> unit

favor player1 player2 forces player2 to give a card to player 1

val check_turn : Gamestate.game -> Player.player -> unit

check_turn g current_player checks if the current_player was attacked and forces said player to take another turn if the player has been attacked, otherwise it will be the next player's turn

val defuse : Gamestate.game -> unit

defuse game negates the losing condition for current player who drew the KABLLAMA! card

val insert_card_at : Deck.card list -> Deck.card -> int -> Deck.card list

insert_card_at card_lst card i inserts card at position i in card_lst

val use_card : Deck.card -> Gamestate.game -> unit

use_card card activates the card and mutates something, returns unit
