Module Deck

module Deck: sig .. end

Representation of dynamic deck data.

This module represents the data stored in deck, including the card remaining and the cards used.

type card_prop = {
   name : string;
   description : string;
}

card_prop is the property of each card, including name and description

type card =
| Skip of card_prop
| Attack of card_prop
| Shuffle of card_prop
| Nope of card_prop
| ProphetLlama of card_prop
| Favor of card_prop
| Defuse of card_prop
| Kabllama of card_prop
| SurfingLlama of card_prop
| EmoLlama of card_prop
| CarrotLlama of card_prop
| HairyLlama of card_prop
| CodingLlama of card_prop

The type of cards there are in this game

val deck : card list

deck is a composition of all the card in the deck, in total, there are 3 Kabllama, 2 Defuse (4 Defuses are already in the starting hands of 4 players), 4 Skip, 4 Shuffle, 4 Attack, 4 Favor, 5 Nope, 5 ProphetLlama, 4 SurfingLlama, 4 EmoLlama, 4 CarrotLlama, 4 CodingLlama, 4 HairyLlama

type t = card list

type t is the representation of the deck

val next_card : t -> card option

next_card t is the next card from t

val shuffle : t -> t

shuffle d takes d and reorders the elements in d, but nothing else is changed

val get_card_name : card -> string

get_card_name card returns the name of card from its properties

val get_card_description : card -> string

get_card_name card returns the description of card from its properties
