Module Player

module Player: sig .. end
type player = {
   name : string;
   mutable turn : int;
   mutable status : bool;
   is_ai : bool;
   mutable points : int;
   mutable hand : Deck.card list;
   mutable next_player : player option;
}

player is an abstract type represent the type of player

val empty_player : player

the empty player

val new_player : string -> player option -> bool -> player

takes in a string for player name, a player option for next player, and a boolean for ai status where true indicates is ai and false indicates is not ai

val add_points : player -> unit

takes in a player and increases their point count by 2

val get_points : player -> int

takes in a player and returns the number of points they have

val deduct_points : player -> unit

takes in a player and deducts their point count by 1

val get_name : player -> string

takes in a player and returns the player name

val get_turn : player -> int

takes in a player and returns an int indicating the number of turns they have to take, where 0 indicates it's not the player's turn, and all other positive integers indicates the number of turns

val update_turn : player -> int -> unit

takes in a player and integer which updates the number of turns the player has to take

val get_status : player -> bool

takes in a player and returns the status, where true indicates that they're still alive and in the game, and false indicates that they're dead and not playing anymore

val get_ai_status : player -> bool

takes in a player and returns a bool, where true indicates that they're an ai, and false indicates that they're not an ai

val update_status : player -> bool -> unit

takes in a player and a bool, which updates the status of the player

val get_hand : player -> Deck.card list

takes in a player, and returns the hand of cards

val add_to_hand : player -> Deck.card -> unit

takes in a player and a card, and updates the hand to contain the additional card

val remove_helper : 'a -> 'a list -> 'a list -> 'a list
val remove_from_hand : player -> Deck.card -> unit

takes in a player and a card, and updates the hand to remove the additional card. If the card does not exist within the player's hand, then no cards will be removed and the player's hand will remain the same

val get_np : player -> player

takes in a player and returns the next player

val update_np : player -> player -> unit

takes in 2 players and updates the first player's next player field to be the second player
